Welcome

Welcome to this Bootstrap demo. Please enjoy your stay! This layout features the following Bootstrap elements:

  • tooltips
  • navbar
  • searchbar
  • form elements
  • dropdown
  • carousel
  • scaffolds
  • buttons
  • media queries
  • icon glyphs
  • blockquote
  • modal

It also uses Lettering.js for the banner.

Please note that this demo is only one page. Also, the search and contact form don't work. They are just styled using Bootstrap css. The purpose of Bootstrap is to provide a style guide with classes that are pre-made so that you don't have to write as much CSS yourself. You don't have to think of your own classes as the Bootstrap classes are well thought out. The documentation is easy to follow as each feature has a visual example and description on how to use it.
